###Shadows-Of-Valor###

Shadows of Valor is a Python text-based adventure game built around player choice, combat mechanics, and progression systems.

This project goes beyond a basic text game and includes multiple gameplay systems such as classes, spells, inventory, and a final boss encounter.

## Features

- Multiple playable classes (Warrior, Mage, Cleric, Ogre)
- Turn-based combat system with critical hits
- Spell system with damage, healing, buffs, and special effects
- Inventory system with health and mana potions
- Random loot and spell unlock system
- Multiple game paths (Forest, Cave, Village)
- Multi-enemy encounters
- Final boss fight with phase-based mechanics and special attacks
- Revive system for certain classes

## How to Run

### Option 1 (Recommended)
Run the executable:
ShadowsOfValor.exe

### Option 2 (Python)
1. Install Python
2. Run:
   python shadows_of_valor.py
